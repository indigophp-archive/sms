**SMS Gateway Abstraction Layer.**


## Install

Via Composer

``` json
{
    "require": {
        "indigophp/sms": "@stable"
    }
}
```


## Usage

``` php
$gateway = new Indigo\Sms\Gateway\MyGateway;

$message = new Indigo\Sms\Message(123456789, 'This is a message', OPTIONAL_SENDER_OR_SENDER_ID);

$gateway->send($message);
```
